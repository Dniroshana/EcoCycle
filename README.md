## 📌 Project Overview
**EcoCycle Connect** is an Android mobile application providing a modern solution for waste management challenges. Developed using Android Studio, this project aligns with the **UN Sustainable Development Goals (SDGs)** to promote environmental sustainability, efficient recycling, and proper waste disposal methods through a "Sustainability Synced" approach.

## ✨ Key Features
* **Schedule Pickups:** Easily book time slots for electronic and other waste collections directly from your location.
* **Rewards System:** Earn points for recycling and track your progress (e.g., Gold Tier) to stay motivated.
* **Waste Inventory & Tracking:** Monitor your recent activities and manage your recyclable inventory.
* **Modern UI/UX:** Built with high-fidelity components designed in Figma.
* **User Authentication:** Secure login and profile management.

## 🛠️ Technology Stack
* **Frontend:** Android Studio (Java / Kotlin)
* **Backend:** Node.js / Express.js / Spring Boot (Update as needed)
* **UI/UX Design:** Figma
* **Version Control:** Git & GitHub
